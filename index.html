Security by design

Полный SSN не должен храниться в браузере или обычной базе

В боевой версии интерфейс отправляет данные напрямую в защищённый backend. В базе сохраняются только последние четыре цифры, токен провайдера, результат, основание проверки, согласие и audit trail.

02Защищённый API

Данные уходят через сервер, а не напрямую из публичного сайта.

03Минимальное хранение
